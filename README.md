# Site Web Benoit

Site web vitrine pour Benoit Taquet Électricité.

## Liens

- **Dépôt GitHub :** [https://github.com/loursCreatif/Site_Web_Benoit](https://github.com/loursCreatif/Site_Web_Benoit)

## Lancer le projet

Pour lancer le site localement, ouvrez un terminal à la racine du projet et exécutez la commande suivante :

```bash
npx -y serve .
```

Cela ouvrira un serveur local (généralement sur http://localhost:3000).
